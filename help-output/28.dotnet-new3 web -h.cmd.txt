.NET Template Creator

ASP.NET Core Empty Template
Author: Microsoft

Usage: dotnet new web [options]

Options:
  -lang|--language  Specifies the language of the template to create.
  -n|--name         The name for the output being created. If no name is specified, the name of the current directory is used.
  -o|--output       Location to place the generated output.
  -h|--help         Displays help for this command.
  -v|--verbose      Set the verbosity level of the command. Allowed values are q[uiet], m[inimal], n[ormal], d[etailed], and diag[nostic]
  --force           Forces content to be generated even if it would change existing files.


ASP.NET Core Empty Options:                                                          
  -f|--framework  The target framework for the project.                              
                      netcoreapp1.0                         
                      netcoreapp1.1 
                      netcoreapp2.0                      
                  Default: netcoreapp2.0                                             

  --skip-restore  If true, skips the automatic restore of NuGet packages for the project on create.
                  Valid values include:
                     True
                     False                                                    
                  Default: false
